# GNN Practice

Repository for practicing Graph Neural Networks (GNN).

## Description
This repository contains code, experiments, and notes related to Graph Neural Networks.  
It is intended for learning, experimenting, and building small projects with GNN models.

## Contents
- `src/` : Source code for models and training scripts
- `data/` : Example datasets or links to datasets
- `notebooks/` : Jupyter notebooks for experiments
- `README.md` : Project documentation

## Goals
- Understand the fundamentals of Graph Neural Networks
- Implement basic GNN architectures
- Run experiments on graph datasets
- Document learning progress

## Requirements
- Python 3.x
- PyTorch / TensorFlow (depending on your setup)
- Additional libraries listed in `requirements.txt`

## How to Run
```bash
git clone https://github.com/kudoshinichi2k5/GNN.git
cd GNN
